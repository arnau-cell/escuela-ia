## If you already know the fundamentals

The usual cross-cutting concepts (what's a model, a database, local vs. cloud, an AI agent...)
still live in [Learn → Concepts](/en/learn/concepts/) — we don't duplicate them here. This Wiki adds
what was missing and brings the tool catalog together in one browsable place.

## About tool ranking

Every tool in the catalog has a "Was this useful?" button — the vote is anonymous (no accounts)
and orders the list within each category. See
[How the ranking works](/en/wiki/how-ranking-works/) for the full formula and the vote's privacy
policy.
