A **sandbox** is an isolated testing space: everything that happens inside it stays inside it,
unable to affect your real computer, your real files, or the internet at large.

## Why it exists

When AI can write and run code, or install things, there needs to be a place where it can make
mistakes without consequences — just like a child can build and knock down sandcastles in a
sandbox without it affecting the rest of the yard. If the AI's code has a bug, or does something
unexpected, it stays contained inside the sandbox.

## Where you'll see it

- When an AI tool "runs code" to answer you (for example, to do a calculation or generate a chart),
  it almost always does so inside a disposable sandbox, not on your computer.
- Agents that can use a terminal or browse files usually operate inside a sandbox for the same
  reason: so that trying something out isn't risky.

## The key idea

A sandbox doesn't make an AI "smarter" — it makes **experimenting with it safe**, both for you and
for whoever builds it.
